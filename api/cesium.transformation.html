train_test_split¶

cesium.transformation.train_test_split(time_series, test_size=0.5, train_size=0.5, random_state=None)¶

Splits input time series into training and test sets.

Samples are stratified based on TimeSeries.target if it is a string (i.e. a class label). See the stratify parameter of sklearn.model_selection for details.

Parameters:

time_series : list of TimeSeries objects

Input time series to be split

test_size : float, optional

Fraction of data to be included in the test set; defaults to 50/50.

training_size : float, optional

Fraction of data to be included in the training set; defaults to 50/50.

random_state : int or RandomState, optional

Random number generator seed or state for shuffling of indices.
Returns:

tuple (list of training set time series, list of test set time series)

transform_ts_files¶

cesium.transformation.transform_ts_files(time_series, transform_type)¶

Apply some transformation to a list of time series and save outputs.

Some transformations will affect each time series individually, whereas some will split the input data into multiple outputs; in either case the time series are returned as a list of lists.

Parameters:

time_series : list of TimeSeries objects

Input time series to be transformed

transform_type : str

Name of transformation to be applied to input time series. Possible values are the keys of TRANSFORM_INFO_DICT.
Returns:

list of lists of TimeSeries objects
